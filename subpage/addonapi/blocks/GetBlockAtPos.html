    integer?, integer? ZVox.AddonAPI.GetBlockAtPos(universe univ, integer x, integer y, integer z)
    View source

    Description

    Gets the voxel ID and voxel state at a certain position given a universe.

    Arguments

    1. universe univ
      Universe to query in
    2. integer x
      X position to query
    3. integer y
      Y position to query
    4. integer z
      Z position to query

    Returns

    1. integer? voxelID
      The voxel ID at that pos, nil if the universe doesn't exist
    2. integer? voxelState
      The voxel state at that pos

    Examples

    Gets the block at 0, 0, 0 from the main universe.

    local univ = ZVox.AddonAPI.GetUniverseByName("main")
local voxID, voxState = ZVox.AddonAPI.GetBlockAtPos(univ, 0, 0, 0)

    Checks for a block being air.

    local univ = ZVox.AddonAPI.GetUniverseByName("main")
local cX, cY, cZ = 1, 1, 1
local voxID, voxState = ZVox.AddonAPI.GetBlockAtPos(univ, cX, cY, cZ)
local airID = ZVox.AddonAPI.GetVoxelID("zvox:air")
if voxID == airID then
	print("The voxel at " .. cX .. ", " .. cY .. ", " .. cZ .. " is air!")
end
